# 学習タイマーシステム（現在開発中）

## 概要

自分の学習タスクと時間を管理するためのWebアプリケーションです。
システム全体を自力で構築するため、現在バックエンドのAPI実装とデータベースの連携をメインに開発を進めています。

## 使用技術 (Tech Stack)

- **バックエンド:** Go
- **データベース:** MySQL
- **インフラ:** Docker
- **フロントエンド:** HTML, CSS, JavaScript

## 開発においてこだわっている点

- 理解負債を残さないことを最優先とし、AIを壁打ち相手として活用しながら、自力でロジックを組み上げることにこだわっています。
- 競技プログラミングでの知見を活かし、計算量やメモリ効率を意識した実装を進めています。
- 現在はフロントとの接続をしようと試みています
